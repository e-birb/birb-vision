## Hikrobot / Hikvision MVS SDK

The `mvs-sys` crate provides Rust FFI declarations for the Hikrobot MVS SDK.

- `crates/bindings/mvs-sys/ffi/mvs.h` is a local wrapper header that includes
  the SDK header `MvCameraControl.h`.
- `crates/bindings/mvs-sys/src/mvs.rs` is generated by `bindgen` from a locally
  installed Hikrobot MVS SDK header.
- The original Hikrobot MVS SDK headers and libraries are not included in this
  repository.
- Hikrobot and Hikvision names, marks, SDKs, headers, libraries, and
  documentation belong to their respective owners.

The generated Rust declarations are included for interoperability with the
Hikrobot MVS SDK. This project is not an official Hikrobot or Hikvision product
and is not endorsed by Hikrobot or Hikvision.
